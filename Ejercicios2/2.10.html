Identifique cada uno de los siguientes objetos de HTML5 como un elmento o un atributo:
  1. a) html
  2. b) width
  3. c) href
  4. d) br
  5. e) h3
  6. f) a
  7. g) src
Respuestas:
  1. a) html elemento
  2. b) width atributo
  3. c) href atributo
  4. d) br elemento
  5. e) h3 elemento
  6. f) a elmento
  7. g) src atributo
